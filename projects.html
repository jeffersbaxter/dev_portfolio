Parkit

ParkIt is a project that assists drivers to find parking in various areas with available driveways. The project members are Jeffers Baxter, Anh Pham, Chyi Yen Wang, and Brian Navaluna. You could almost think of project like "The AirBNB for parking". The application incorporates map and payment APIs to help those struggling to find parking spaces in their areas.

StreamTV

I used the Youtube API to display videos based on search terms that the user enters. A click on a link will take the user to the show page of that video, where the user can either view the video or click on a related video.

Teeny Links

Teeny Links is a URL shortener that I created with the use of Node.js, Express, Postgresql, Sequelize and embedded javascript.

Four-Down Football

You have four downs to score! Failure to do so will give your opponent a chance. Press the space bar to move the ball. The ball will move based on how long the space bar is held down, but don't hold it for too long or it will result in a turnover.
